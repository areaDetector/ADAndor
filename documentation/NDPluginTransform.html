Overview

This plugin allows up to four transformations (rotations and flips) of the NDArray. The for each of the four transforms the user can select from:

  • None
  • RotateCW90
  • RotateCCW90
  • Rotate180
  • Flip0011
  • Flip0110
  • FlipX
  • FlipY
